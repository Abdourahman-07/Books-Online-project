# Books-Online-project
OpenClassrooms project

## Pour une exécution réussite du code :
Avant d'exécuter le code, veillez à : 
- Mettre en place un environnement virtuel en entrant la commande (dans le répertoire Books-Online-project): python -m venv env
- Pour activer l'environnement taper la commande env\Scripts\activate.bat sous Windows ou env\Scripts\activate sous un autre os ou sous Powershell.
- Télécharger tous les packages cités dans le fichier requirements.txt.
- Lancer le fichier main.py avec la commande "python main.py".

## Résultat attendu : 
- un dossier csv_books avec à l'intérieur un fichier csv par catégorie de livres du site BookToScrape, dans ces fichiers on y retrouve les données de leurs livres respectifs (url, UPC, prix, ...)
- un dossier images rempli de sous-dossiers représentant les catégories de livres, dans chaque sous-dossier les images associées de tous les livres de la catégorie correspondante.
