# Books-Online-project
OpenClassrooms project

## Important
Avant d'exécuter le code, veillez à : 
- Mettez en place un environnement virtuel en entrant la commande (dans le répertoire Books-Online-project): python -m venv env
- Pour activer l'environnement tapez la commande env\Scripts\activate.bat sous Windows ou env\Scripts\activate sous un autre os ou sous Powershell.
- Télécharger tous les packages cités dans le fichier requirements.txt.
